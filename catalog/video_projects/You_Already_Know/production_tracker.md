## Phase 1: Pre-Production

### Creative
- [ ] Choose video concept direction (see storyboard.md)
- [ ] Finalize storyboard — lock section-by-section visuals
- [ ] Build detailed shot list with lens/angle specs
- [ ] Create mood board (reference images, color palette, vibe clips)
- [ ] Storyboard review — walk through with anyone involved

### Casting & Crew
- [ ] Confirm on-screen talent (artist + partner/actors?)
- [ ] Director / DP (self-directed or hiring?)
- [ ] Camera operator (if not self)
- [ ] Lighting assistant
- [ ] Hair/makeup/wardrobe (if applicable)

### Locations
- [ ] Scout location options
- [ ] Test lighting at each location (golden hour times, interior light)
- [ ] Secure permissions / permits if needed
- [ ] Plan backup locations (weather contingency)

### Gear & Logistics
- [ ] Camera body + lenses confirmed
- [ ] Gimbal / stabilizer
- [ ] Lighting kit (if interior)
- [ ] Audio playback system (speaker for lip-sync on set)
- [ ] Memory cards + backup drives
- [ ] Batteries charged
- [ ] Props list
- [ ] Wardrobe pulls — fits color palette?

### Schedule
- [ ] Lock shoot date(s)
- [ ] Create call sheet with times & locations
- [ ] Build shot schedule (order by location, not song order)

## Phase 2: Production (Shoot Day)

### On Set
- [ ] Playback track loaded and tested
- [ ] Color chart / white balance reference shot
- [ ] Slate each setup (section name + take #)
- [ ] Performance takes — full song, multiple angles
- [ ] Narrative takes — per storyboard
- [ ] B-roll beauty shots
- [ ] Safety coverage — wide master of each location
- [ ] Review footage on set before wrapping each location

### Sections to Capture
- [ ] Intro shots (solo, window light, details)
- [ ] Verse 1 shots (lonely → warmth transition)
- [ ] Chorus performance + narrative (PUNCH word moments)
- [ ] Instrumental b-roll
- [ ] Verse 2 shots (intimate, hands, dancing)
- [ ] Bridge shots (night/lights, dreamy, reflections)
- [ ] Final Chorus (full energy, all locations)
- [ ] Outro (pull-back, fade to warm)

---

## Phase 3: Post-Production

### Edit
- [ ] Ingest & organize footage (by section)
- [ ] Rough cut — assemble to song timeline
- [ ] Fine cut — timing, transitions, pacing
- [ ] Add slow-motion / speed ramps
- [ ] Overlay effects (snow particles, light leaks, etc.)

### Color
- [ ] Apply base color grade (cool → warm arc)
- [ ] Match shots within each section
- [ ] Final color pass — polish

### Finishing
- [ ] Titles / lower thirds (if any)
- [ ] Final export — master resolution
- [ ] Export social cuts (vertical 9:16, square 1:1, teaser)
- [ ] Upload to hosting / distribution

### Release
- [ ] Thumbnail design
- [ ] Write video description / credits
- [ ] Plan release date & promotion
- [ ] Upload to YouTube / Vimeo
- [ ] Share across socials
- [ ] Submit to music video blogs / playlists (optional)
